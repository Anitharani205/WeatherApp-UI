 Weather Dashboard – Delhi (20 Years)
 
Overview
A simple web application that fetches and displays 20 years of weather data for Delhi using a public Weather API.

Features
● View yearly weather data
● Search by year
● Filter by minimum and maximum temperature
● Sort by Year, Temperature, and Humidity
● Delete records from the table
● Clean and responsive UI 

Tech Stack
● HTML
● CSS
● JavaScript
● Fetch API
● Public Weather API
